¿Qué es HTML?

HTML es el lenguaje estándar que se emplea para la creación y la estructuración de páginas web.
Por ejemplo, cada página web que visitas, detrás tiene un código estructurado con HTML, ya que este le indica al navegador cómo tiene que mostrar la información.

Ejemplos de formato de texto

Para escribir en negrita tendrás que hacer uso de <b>"el texto"</b>

Para escribir en cursiva tendrás que hacer uso de <i>"el texto"</i>

Para escribir en subrayado tendrás que hacer uso de <u>"el texto"</u>

Para escribir en subíndice tendrás que hacer uso de <sub>"el texto"</sub>

Para escribir en superíndice tendrás que hacer uso de <sup>"el texto"</sup>

Listas en HTML

Para crear listas ordenadas utilizaremos <ol>"lista"</ol>

En cambio, para una lista desordenada emplearemos <ul>"lista"</ul>

Y para añadir un elemento a la lista hay que hacer uso de <li>"texto"</li>

Ejemplo de una lista ordenada:

  1. Aplicaciones Web
  2. Digitalización
  3. Sostenibilidad

Ejemplo de una lista desordenada

  • Gimnasio
  • Fútbol
  • Videojuegos

Para definiciones utilizaremos <dl>"definiciones"</dl>

Para el término a definir emplearemos <dt>"término"</dt>

Y para la definición del término emplearemos <dd>"definción"</dd>

Enlaces en HTML

Si queremos añadir links a nuestra página web, tendremos que utilizar <a>href="url">"texto"</a>

También podemos añadir un link que nos diriga a un apartado en específico de nuestra página web. Para ello tendremos que utilizar añadirle un "id" al apartado que queramos, y posteriormente

Es decir, por ejemplo si queremos que añadir un ID a un h1, se realizaría de la siguiente manera. Locarizaremos el h1 que queramos y modificaremos la etiqueta, quedaría tal que así:

<h1 id="palabra clave">"título"</h1>

Tendríamos que utilizar lo siguiente para añadir un link a un ID:

<a href="#'palabra clave'">"nombre del link"</a>

Imágenes en HTML

En cambio, si queremos añadir imágenes almacenadas en nuestro equipo, tendremos que utilizar <img src="'dirección de la imagen'" alt="'texto alternativo'"

Tablas en HTML

Para crear una tabla en nuestra página web deberemos emplear <table>"contenido de la tabla"</table>

Para cada fila de nuestra tabla deberemos utilizar <tr>"contenido de la fila"</tr>

Y para cada columna de la fila deberemos utilizar <td>"contenido de la columna"</td>

Pero si queremos que el texto sea un encabezado utilzaremos <th>"contenido de la columna"</th>
